--------------------------------------  

.. raw:: html

 
 <p style="text-align:justify "><a href="#id4.1.11"> 4.1.11</a> <span id="id4.1.11"> 在实际检测中，加前置放大器后，超过5 m 距离所测超声波数值漂移较大，不稳定，因此本条规定超声波测距不宜超过5 m。</span></p>

4.4 裂缝检测
--------------------------------------  

.. raw:: html

 <p style="text-align:justify "><a href="#id4.4.2"> 4.4.2</a> <span id="id4.4.2"> 裂缝宽度是验算混凝土结构安全性的依据，目前国内外市场出现了裂缝宽度测定仪，提高了测试精度。</span></p> 
 <p style="text-align:justify "><a href="#id4.4.3"> 4.4.3</a> <span id="id4.4.3"> 先将不跨缝测试的混凝土声速<i>v</i><sub>p</sub>计算出来，再计算求得裂缝深度，提高测试结果的准确性。</span></p> 
 <p style="text-align:justify;text-indent:2em;" > 本条参考了现行协会标准《超声法检测混凝土缺陷技术规程》（ CECS 21-2000 ）的有关规定。</p> 

4.5 表面损伤层厚度检测
--------------------------------------  

.. raw:: html

 <p style="text-align:justify "><a href="#id4.5.3"> 4.5.3</a> <span id="id4.5.3"> 混凝土表面损伤层检测，一般是将换能器放在同一测试面上进行单面平测，这种测试方法超声波接收信号较弱，换能器频率越高，接收信号越弱。因此，为保证信号具有一定幅值，读数准确，宜选用频率较低的平面换能器。</span></p>

4.8 钢管混凝土缺陷检测
--------------------------------------  

.. raw:: html

 
 <p style="text-align:justify "><a href="#id4.8.3"> 4.8.3</a> <span id="id4.8.3"> 由于超声波在钢材中的声速远快于混凝土声速，如果钢管尺寸过小，声波很可能完全沿钢管传播，就不能反映钢管内混凝土的质量情况。</span></p> 
 
4.9 地连墙混凝土缺陷检测
--------------------------------------  

.. raw:: html

 
 <p style="text-align:justify "><a href="#id4.9.2"> 4.9.2</a> <span id="id4.9.2"> 地连墙一般属隐蔽工程，目前采用施工时预埋声测管的方式对墙体质量进行检测的需求加大，所以推荐采用预埋声测管方法。</span></p> 
 <p style="text-align:justify "><a href="#id4.9.3"> 4.9.3</a> <span id="id4.9.3"> 单地连墙槽段长度一般在（4~8）m 不等，因此可以根据实际情况在所选取槽段上均匀埋置声测管。</span></p>
